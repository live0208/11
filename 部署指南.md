# 光予 — 部署 & APK 生成

## 一、部署到 GitHub Pages（获取 HTTPS）

1. 打开 github.com/new — 仓库名填 `guangyu`，Public，点创建
2. 点 "uploading an existing file"
3. 把 app/ 里所有文件拖进去，点 Commit
4. Settings → Pages → Source 选 main → Save
5. 30秒后得到地址：`https://用户名.github.io/guangyu`

## 二、生成 APK

1. 打开 pwabuilder.com
2. 输入上面的 HTTPS 地址
3. 点 Package for Stores → Android → Download APK
4. 传到手机安装即可

---

手机浏览器打开 `http://172.27.224.228:8080` 也可以直接添加到主屏幕。
